# 🏥 스마트 보건실 (Smart Health Log System)
> **보건실 방문 접수 및 대기 현황을 실시간으로 관리하는 웹 애플리케이션입니다.**

이 프로젝트는 학교 보건실의 혼잡도를 줄이고, 선생님이 학생들의 증상을 한눈에 파악하여 효율적으로 처치할 수 있도록 돕기 위해 제작되었습니다.

---

## ✨ 주요 기능

### 1. 학생용 (접수 및 대기)
- **실시간 대기 현황**: 현재 대기 인원과 예상 대기 시간을 메인 위젯에서 즉시 확인 가능
- **간편 접수**: 학번, 이름, 증상을 선택하여 빠르게 접수 (개인정보 수집 동의 포함)
- **날씨 정보**: 실시간 날씨 정보를 통해 건강 관리에 유의할 수 있도록 안내

### 2. 관리자용 (선생님 페이지)
- **대기 명단 관리**: 접수된 학생들을 실시간 리스트로 확인 및 처치 완료 처리
- **데이터 관리**: 방문 기록을 CSV(엑셀) 파일로 다운로드하여 통계 관리 가능
- **개인정보 보호**: 비밀번호 잠금 기능 및 민감 정보 보호 설계

## 🔒 보안 및 개인정보 유의사항
- **최소 정보 수집**: 학번, 이름 등 처치에 필요한 최소한의 정보만 수집합니다.
- **접근 제한**: 관리자 페이지는 지정된 비밀번호를 통해서만 접근할 수 있습니다.
